CODE TRANSFORMERS

This project presents a sensor-based system for enhancing railway track monitoring and safety. Acoustic emission sensors detect and locate cracks, ultrasonic sensors measure rail distances, and PIR sensors detect humans or animals on tracks. GPS provides precise location data, and GSM-R enables real-time communication with the base station. A microcontroller processes data and transmits it via satellite or GPRS. This solution addresses key challenges in current systems, improving crack detection, track maintenance, and accident prevention.
